From your backend (app/api/ directory), the endpoints are:

Logs Endpoints (app/api/logs.py):
POST /logs/: Ingest a custom log.
GET /logs/: Retrieve all logs.
POST /logs/mock: Generate and store a mock log (already tested).
Rules Endpoints (app/api/rules.py):
POST /rules/generate: Generate a detection rule using Groq based on existing logs.
GET /rules/: Retrieve all rules.
Alerts Endpoints (app/api/alerts.py):
POST /alerts/detect: Detect threats using a specific rule and generate alerts.
GET /alerts/: Retrieve all alerts.
